 This plugin will not allow the user to access login pages after logout when press the browser back button 

== Description ==

Generally user when logout from wordpress website and after they press browser back button then they can see previous login pages(but accessable).
This plugin eliminate this thing.

== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `wps-safe-logout` and click 'Install Now' or click on the `upload` link to upload `wps-safe-logout.zip`
4. Click on `Activate plugin`
